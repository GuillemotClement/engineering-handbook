# engineering-handbook
Centralized notes and references about software engineering, systems, and modern web development.

The project is host by netlify on `https://https://engineering-handbook.netlify.app/`

## Stack 

- **Mkdocs**: use for generate HTML from folder `/docs` with markdown file
- **Netlify**: allow to share the projet for use anywhere 
- **Obsidian**: for write some new content
