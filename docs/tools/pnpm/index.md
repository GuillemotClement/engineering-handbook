# PNPM 
[pnpm](https://pnpm.io/fr/)

Remplace `npm`. 

--- 

## Commande de base

| `pnpm` | `npm` | action | 
| --- | --- | --- |
| `pnpm install` | `npm install` | installation des dépendances du projet |
| `pnpm i <package>`| `npm i <package>`| installer d'une dépendance | 
| `pnpm <cmd>` | `npm run <cmd>` | lancement d'un script définis dans le `package.json`|
