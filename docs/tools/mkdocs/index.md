# Mkdocs 

Permet de générer des sites rapidement en créant des pages à l'aide du format `markdown`.

## Local 

Pour lancer le serveur de dev, il faut utiliser l'environnement virtuel avec `venv`. Avec `fish`, il faut utiliser cette commande ` source .venv/bin/activate.fish`.

Cela permet d'activer l'environnement virtuel, et d'utiliser les dépendances correctement. Cette commande est à lancer pour pouvoir lancer le serveur de dev.

Pour lancer le server `mkdocs`, et tester que le rendu est correct 

```shell
mkdocs serve
```
