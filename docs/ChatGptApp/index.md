# ChatGpt App

Un ChatGPT App est une application web qui s'exécute à l'intérieur de ChatGPT et possède sont propre widget UI. 

L'applicatin fournit à ChatGPt un ensemble d'outils (fonctions) et de données, et est enregistrée dans le catalogue d'appliction `App Store`.
Elle combine une interface conversationnelle avec un UI graphique et une logique backend, en communiquant avec la plateforme GPT via des protocoles standardisés comme `MCP`.
