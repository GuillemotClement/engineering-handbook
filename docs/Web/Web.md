## Code HTTP

| Code | Nom                  | Raison                                                                                                   |
| ---- | -------------------- | -------------------------------------------------------------------------------------------------------- |
| 422  | Unprocessable Entity | Le contenu de la requête ne respect pas les règles métiers (champ manquant, format email incorrect ... ) |
| 409  | Conflict             | La requête entre en collision avec une donnée existante (email déjà utilisé, etc)                        |
